# 🇸🇴 SAKO-99 GHOST CLEANER V3 🛡️

Mashiinkan waa kii ugu dambeeyay ee loogu talagalay inuu si toos ah u tirtiro Logs-ka ciyaaraha (Anti-Ban Support). Wuxuu leeyahay nidaam doorasho (Interactive Menu) iyo farriimo ku hagaya isticmaalka Shizuku.

This is the latest standalone tool designed to automatically delete game security logs (Anti-Ban Support). It features an interactive menu and Shizuku guidance.

---

## 🚀 Sida loo kiciyo (How to Run)

Nuqul ka qaado amarkan hoose si aad hal mar ugu rakibto uguna kiciso Termux-kaaga:
Copy and paste this command to install and run it instantly in Termux:

```bash
pkg update && pkg upgrade -y && pkg install python git -y && git clone [https://github.com/Maxamedxas/SAKO-99-ULTIMATE.git](https://github.com/Maxamedxas/SAKO-99-ULTIMATE.git) && cd SAKO-99-ULTIMATE && python cleaner_v3.py
